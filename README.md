# Global Movie Data Analysis Dashboard (Power BI)

## Project Overview
This project presents a comprehensive Movie Data Analysis Dashboard built using Power BI. The dashboard analyzes global movie data to uncover insights related to revenue generation, movie ratings, genre popularity, language distribution, and trends over time.

The goal of this project is to demonstrate how data visualization and business intelligence tools can be used to extract meaningful insights from large datasets. The dashboard allows users to interactively explore movie industry trends and understand the factors influencing movie performance.

## Problem Statement
The movie industry produces thousands of films every year across different genres and languages. However, analyzing large volumes of movie data to identify patterns, trends, and successful genres can be challenging.

This project aims to analyze movie data to identify key insights such as:
- Which genres generate the highest revenue
- How movie production has evolved over time
- The relationship between movie ratings and revenue
- Popular languages and their contribution to revenue

## Dataset Description
The dataset contains information about movies across different years and includes attributes such as:

- Movie Title
- Genre
- Release Year
- Original Language
- Runtime
- Vote Count
- Average Rating
- Revenue

The dataset contains approximately 9,800+ movies and covers multiple languages and genres.

## Tools and Technologies
- Power BI
- Power Query
- Data Modeling
- DAX (Data Analysis Expressions)
- Data Visualization

## Data Preparation
The dataset was prepared using Power Query with the following steps:

- Removed duplicate records
- Handled missing values
- Converted columns to appropriate data types
- Created calculated measures using DAX
- Structured the dataset for efficient data modeling

## Dashboard Pages

### Overview Page
Provides a high-level summary of movie data including:
- Revenue by Genre
- Movies released by Year
- Revenue trends over time
- Genre distribution

### Strategy Page
Focuses on language and genre insights including:
- Movies by Genre
- Rating by Language
- Revenue by Language
- Movie distribution by language

### Performance Page
Analyzes movie performance including:
- Top revenue generating movies
- Rating vs Revenue analysis
- Vote count vs Revenue
- Runtime trends over time
- Runtime by Genre

## Key Insights
- Drama, Comedy, and Action genres contribute the largest share of movies and revenue.
- Movie production has significantly increased after the year 2000.
- Higher vote counts generally correlate with higher movie popularity.
- English language movies dominate the global movie dataset.
- Certain genres consistently receive higher audience ratings.
